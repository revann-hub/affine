# affine
This code implements an Affine cipher for encrypting and decrypting text. The compute_gcd function finds the greatest common divisor of two numbers using subtraction. The generate_keys function creates random key pairs for encryption, ensuring the first key is coprime with 26 for mathematical validity. The encode_message function encrypts user input by asking whether they want to provide their own keys or have them automatically generated, then applies the formula (first_param * position + second_param) mod 26 to each letter while preserving case and non-alphabetic characters. The decode_message function decrypts text by asking for the original keys, calculating the modular inverse of the first key, and applying the reverse formula to recover the original text. Both functions handle lowercase and uppercase letters separately while leaving other characters unchanged.
